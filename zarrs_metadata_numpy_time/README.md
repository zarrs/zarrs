NumPy `datetime64`/`timedelta64` types for the [`zarrs`](https://crates.io/crates/zarrs) Rust crate.

This crate provides the shared type definitions used by both `zarrs_metadata_ext` and `zarrs_data_type` for NumPy temporal data types.
